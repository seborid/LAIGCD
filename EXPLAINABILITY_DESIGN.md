# Deepfake人脸检测系统可解释性设计方案

## 项目定位

**核心目标**：轻量化、可解释的**Deepfake人脸检测系统**

**关键特征**：
- 目标：AI生成的人脸图像检测（非通用物体检测）
- 技术：空域 + 频域综合分析
- 要求：轻量化（~800K参数）、可解释性

## 第一阶段：检测器 + 热力图生成

### 输出设计

| 输出 | 格式 | 用途 |
|-----|------|-----|
| 真假判断 | float (0-1) | 二分类概率 |
| 置信度 | float (0-100%) | 判断可信度 |
| 空域热力图 | [H, W] PNG | 可视化模型关注区域 |
| 频域热力图 | [H, W] PNG | 可视化频域异常位置 |
| 原型激活 | [16] list | 各原型激活程度 |

### 空域热力图方案

**方案1：CLIP Attention Rollout** ⭐推荐
- 基于ViT的多层注意力累积
- 无需额外训练
- 计算高效

**方案2：Grad-CAM**
- 基于梯度的类激活映射
- 需要反向传播
- 更精确但稍慢

**方案3：原型注意力投影**
- 计算每个patch与原型的相似度
- 可解释性强（每个原型对应语义）
- 适合展示"模型为什么关注这里"

### 频域热力图方案

**方案1：SRM滤波响应** ⭐推荐
- 30个高通滤波器的响应聚合
- 直接来自频域模块
- 无额外计算

**方案2：FFT局部能量**
- 分块FFT计算高频能量
- 直观展示频域异常
- 需要额外计算

**方案3：学习的频域注意力**
- 在频域模块中添加空间注意力分支
- 可训练优化
- 需要修改模型

## 第二阶段：解释生成

### 技术路径

**主方案：FakeVLM零样本推理**

输入格式：
```json
{
    "image": ["original.jpg", "spatial_heatmap.png", "freq_heatmap.png"],
    "conversations": [
        {
            "from": "human",
            "value": "你是deepfake检测专家。分析以下图像：第一张是人脸原图，第二张是空域注意力热力图（红色表示高关注区域），第三张是频域异常热力图（亮色表示频域伪影）。请判断是否为AI生成的人脸，并详细说明依据。"
        }
    ]
}
```

### Prompt工程

**基础Prompt模板**：
```
你是AI生成人脸检测专家。分析以下检测结果：

原图：待检测的人脸图像
空域热力图：显示模型关注的异常区域（红色=高关注）
频域热力图：显示频域伪影位置（亮色=频域异常）

请从以下角度分析：
1. 面部特征自然度（发际线、瞳孔、眉毛、光影等）
2. 空域热力图的高关注区域是否存在异常
3. 频域热力图的异常模式是否符合AI生成特征
4. 综合判断并给出置信度
```

**Few-shot增强**（如零样本效果不佳）：
```
示例1：
[图组A]
分析：空域热力图显示眼部和嘴部区域高关注，这些区域存在模糊；频域热力图显示整体高频噪声强。这是AI生成的deepfake。
结论：AI生成（置信度85%）

示例2：
[图组B]
分析：空域热力图分布均匀，无异常聚焦；频域热力图显示正常频域分布。
结论：真实人脸（置信度92%）

现在分析这张图：[待检测图]
...
```

### 可行性分析

| 要点 | 状态 | 说明 |
|-----|------|-----|
| FakeVLM支持多图 | ✅ | LLaVA支持image为list |
| 理解热力图 | ⚠️ | 需要验证，伪彩色图可能需要适应 |
| 零样本效果 | ⚠️ | 需要prompt工程测试 |
| 无需训练 | ⚠️ | 优先尝试零样本，备选少量LoRA微调 |

## 备选方案

### 备选1：规则+模板（轻量级）

如FakeVLM效果不佳或资源受限：

```python
def generate_explanation(confidence, proto_activations, freq_stats, spatial_attn):
    """基于规则和模板生成解释"""

    # 1. 识别主要激活原型
    top_protos = proto_activations.argsort(descending=True)[:3]

    # 2. 原型语义映射
    proto_semantics = {
        0: "高频噪声模式",
        1: "频域周期性伪影",
        2: "边缘模糊异常",
        3: "纹理不连续",
        4: "颜色分布异常",
        15: "面部特征不协调"
    }

    # 3. 空域高关注区域分析
    spatial_peaks = find_peaks(spatial_attn)
    region_analysis = analyze_regions(spatial_peaks)  # "眼部区域", "嘴部区域"等

    # 4. 生成解释
    explanation = f"""
这张图片{'很可能是' if confidence > 0.8 else '可能是'}AI生成的deepfake人脸（置信度{confidence:.0%}）。

主要依据：
"""

    for proto_idx, activation in top_protos:
        if activation > 0.5:
            explanation += f"• {proto_semantics[proto_idx]}（激活度{activation:.0%}）\n"

    if freq_stats['high_freq_anomaly']:
        explanation += "• 频域分析显示高频能量分布异常\n"

    if region_analysis:
        explanation += f"• 空域分析发现{region_analysis}存在异常\n"

    return explanation
```

**输出示例**：
```
这张图片很可能是AI生成的deepfake人脸（置信度87%）。

主要依据：
• 高频噪声模式（激活度85%）
• 面部特征不协调（激活度72%）
• 频域分析显示高频能量分布异常
• 空域分析发现眼部和嘴部区域存在异常
```

## 关键技术点

### 原型语义标注（16个原型）

| ID | 语义 | 对应特征 |
|----|------|---------|
| 0 | 高频噪声模式 | SRM高频滤波响应 |
| 1 | 频域周期性伪影 | DCT周期性模式 |
| 2 | 边缘模糊异常 | 边缘高频缺失 |
| 3 | 纹理不连续 | 纹理频谱突变 |
| ... | ... | ... |
| 15 | 面部特征不协调 | 综合特征 |

### 热力图配色方案

- **空域热力图**：jet/rainbow（红=高关注）
- **频域热力图**：viridis/magma（亮=高频异常）

## 风险与应对

| 风险 | 应对 |
|-----|------|
| FakeVLM无法理解热力图 | 备选规则+模板方案 |
| 零样本效果差 | Few-shot或少量LoRA微调 |
| 热力图质量不够 | 优化热力图生成方法 |
| 推理速度慢 | 缓存或使用量化模型 |
